  <action>
    Create `src/snapshot.rs` with the following:

    1. **Snapshot struct** (per spec §10/KB-D17, decision D-03):
       ```rust
       #[derive(Debug, Clone, Serialize, Deserialize)]
       pub(crate) struct Snapshot {
           pub(crate) timestamp: String,  // ISO 8601 UTC
           pub(crate) handles: HandleCounts,
           pub(crate) edges: EdgeCounts,
           pub(crate) states: HashMap<String, usize>,
           pub(crate) obligations: ObligationCounts,
           pub(crate) diagnostics: DiagnosticCounts,
           pub(crate) namespaces: HashMap<String, NamespaceStats>,
       }
       ```
       With sub-structs:
       - `HandleCounts { total, active, frozen }` (all usize)
       - `EdgeCounts { total }` (usize)
       - `ObligationCounts { outstanding, discharged, mooted }` (all usize)
       - `DiagnosticCounts { errors, warnings }` (all usize)
       - `NamespaceStats { total, open, resolved, deferred }` (all usize — open=no status, resolved=terminal, deferred=active but stale beyond threshold)

    2. **build_snapshot function**: Takes `&DiGraph`, `&Lattice`, `&AnnealConfig`, `&[Diagnostic]`, and returns `Snapshot`. Iterates graph.nodes() to count handles by active/terminal, group by status. Counts obligations by iterating linear namespace labels (same logic as check_linearity). Counts diagnostics by severity. Computes namespace stats by grouping Label handles by prefix.

    3. **JSONL I/O** (per spec §15.2, decisions D-01, D-02):
       - `append_snapshot(root: &Utf8Path, snapshot: &Snapshot) -> anyhow::Result<()>`: Create `.anneal/` directory with `fs::create_dir_all`. Open `history.jsonl` with `OpenOptions::new().create(true).append(true).open()`. Serialize with `serde_json::to_vec`, push `b'\n'`, single `write_all`. No BufWriter, no fsync.
       - `read_history(root: &Utf8Path) -> Vec<Snapshot>`: If file missing, return empty Vec. Read with `BufReader::lines()`, `filter_map` with `serde_json::from_str`, warn to stderr on bad lines (eprintln!).

    4. **Convergence summary** (per spec §10.1/KB-D18, decisions D-05, D-06):
       ```rust
       #[derive(Debug, Clone, Serialize)]
       pub(crate) enum ConvergenceSignal { Advancing, Holding, Drifting }

       #[derive(Debug, Clone, Serialize)]
       pub(crate) struct ConvergenceSummary {
           pub(crate) signal: ConvergenceSignal,
           pub(crate) detail: String,
       }
       ```
       - `compute_convergence_summary(current: &Snapshot, previous: &Snapshot) -> ConvergenceSummary`: Compare handle counts (frozen delta = resolution), obligations (outstanding delta), and states histogram shift. Score: resolution_gain = current.handles.frozen - previous.handles.frozen. creation_gain = current.handles.total - previous.handles.total. obligations_delta = current.obligations.outstanding as i64 - previous.obligations.outstanding as i64. If resolution_gain > creation_gain && obligations_delta <= 0 => Advancing. If creation_gain > resolution_gain || obligations_delta > 0 => Drifting. Else => Holding.
       - `latest_summary(root: &Utf8Path, current: &Snapshot) -> Option<ConvergenceSummary>`: Reads history, if empty returns None (D-06), else calls compute against last entry.

    5. Add `mod snapshot;` to `src/main.rs` after the existing module declarations (after `mod resolve;`).

    All structs must derive `Serialize` (and `Deserialize` for Snapshot). Use `chrono::Utc::now().to_rfc3339()` for timestamp.
