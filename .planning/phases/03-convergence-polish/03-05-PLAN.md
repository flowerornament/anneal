<objective>
Implement the `anneal diff` command for graph-level change tracking.

Purpose: Diff answers "what changed while I was away?" -- the question every arriving agent asks. Three modes: compare against last snapshot (default), against a time-based reference (--days), or against a git ref for true structural diff. This is the convergence tracking feature that connects sessions across time.

Output: `anneal diff` command with three reference modes, producing structured change summaries per spec section 12.8.
</objective>

       ```rust
       #[derive(Serialize)]
       pub(crate) struct HandleDelta {
           pub(crate) created: i64,     // current.total - previous.total
           pub(crate) active_delta: i64,
           pub(crate) frozen_delta: i64,
       }

       #[derive(Serialize)]
       pub(crate) struct StateChange {
           pub(crate) state: String,
           pub(crate) previous_count: usize,
           pub(crate) current_count: usize,
           pub(crate) delta: i64,
       }

       #[derive(Serialize)]
       pub(crate) struct ObligationDelta {
           pub(crate) outstanding_delta: i64,
           pub(crate) discharged_delta: i64,
           pub(crate) mooted_delta: i64,
       }

       #[derive(Serialize)]
       pub(crate) struct EdgeDelta {
           pub(crate) total_delta: i64,
       }

       #[derive(Serialize)]
       pub(crate) struct NamespaceDelta {
           pub(crate) prefix: String,
           pub(crate) total_delta: i64,
           pub(crate) open_delta: i64,
           pub(crate) resolved_delta: i64,
       }
       ```

    3. **diff_snapshots function** (core diff logic):
       ```rust
       fn diff_snapshots(current: &Snapshot, previous: &Snapshot) -> DiffOutput
       ```
       - HandleDelta: subtract previous from current for total, active, frozen.
       - StateChanges: union of all state keys from both snapshots. For each: compute delta. Only include states where delta != 0.
       - ObligationDelta: subtract previous from current.
       - EdgeDelta: subtract previous.edges.total from current.edges.total.
       - NamespaceDelta: union of namespace keys. For each: compute deltas on total, open, resolved. Only include namespaces with non-zero deltas.
       - Set reference and has_history appropriately.

    4. **print_human implementation** matching spec §12.8:
       ```
       Since {reference}:
         Handles: {delta} ({active_delta} active, {frozen_delta} frozen)
         State changes: {state}: {prev} -> {curr} ({delta:+})
         Obligations: {outstanding_delta:+} outstanding, {discharged_delta:+} discharged, {mooted_delta:+} mooted
         Edges: {delta:+}
         Namespaces: {prefix}: {delta:+} total ({open_delta:+} open, {resolved_delta:+} resolved)
       ```
       Use `+` prefix for positive deltas, no prefix for negative (just the minus sign). When has_history is false, print "No history available. Run anneal check or anneal status to create a baseline snapshot."

    5. **find_snapshot_by_days helper**:

       ```rust
       fn find_snapshot_by_days(history: &[Snapshot], days: u32) -> Option<&Snapshot>
       ```
       Parse timestamps with `chrono::DateTime::parse_from_rfc3339`. Find the snapshot whose timestamp is closest to `Utc::now() - Duration::days(days)`. Return None if history is empty.

    6. **build_graph_at_git_ref helper** (D-09):
       ```rust
       fn build_graph_at_git_ref(

           git_ref: &str,
       ) -> anyhow::Result<Snapshot>
       ```

       Implementation:
       - Create a temp directory with `std::env::temp_dir().join("anneal-diff-{random}")`
       - Shell out to `git -C {root} archive {git_ref} | tar -x -C {temp_dir}` to extract files at that ref. Use `std::process::Command::new("sh").arg("-c")` with the piped command.
       - If git command fails (non-zero exit), return error with the stderr message.
       - Run the full anneal pipeline on the temp dir: `config::load_config`, `parse::build_graph`, `resolve::resolve_all`, `lattice::infer_lattice`, `checks::run_checks`.
       - Build a Snapshot from the result via `snapshot::build_snapshot`.
       - Clean up temp dir with `std::fs::remove_dir_all`.
       - Return the snapshot.

    7. **cmd_diff function**:
       ```rust
       pub(crate) fn cmd_diff(
           root: &Utf8Path,
           current_snapshot: &Snapshot,
           days: Option<u32>,
           git_ref: Option<&str>,
       ) -> anyhow::Result<DiffOutput>
       ```
       - If `git_ref` is Some: call `build_graph_at_git_ref`, diff current against the reconstructed snapshot.
       - If `days` is Some: read history, find closest snapshot by days, diff against it.
       - If neither (default): read history, take the last entry, diff against it.
       - If no history and no git ref, return DiffOutput with has_history=false and zero deltas.

       ```rust
       Some(Command::Diff { days, ref git_ref }) => {
           let (unresolved_refs, section_ref_count) =
               collect_unresolved(&result.pending_edges, &node_index);
           let unresolved_owned: Vec<parse::PendingEdge> = unresolved_refs
               .iter()
               .map(|e| parse::PendingEdge {
                   source: e.source,
                   target_identity: e.target_identity.clone(),
                   kind: e.kind,
                   inverse: e.inverse,

               .collect();

               cli::print_json(&output)?;
       ```
