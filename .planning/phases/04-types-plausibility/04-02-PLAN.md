**Step 4: Add W004 diagnostic to checks.rs**

Add a new function in checks.rs after check_existence:

```rust
/// W004: Implausible frontmatter values that were filtered before resolution.
fn check_plausibility(implausible_refs: &[crate::parse::ImplausibleRef]) -> Vec<Diagnostic> {
    implausible_refs
        .iter()
        .map(|r| Diagnostic {
            severity: Severity::Warning,
            code: "W004",
            message: format!(
                "implausible frontmatter value {:?} ({})",
                r.raw_value, r.reason
            ),
            file: Some(r.file.clone()),
        })
        .collect()
}
```

Add `use crate::parse::ImplausibleRef;` to checks.rs imports.
