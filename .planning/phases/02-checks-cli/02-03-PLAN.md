        println!("{json}");
        Ok(())
    }

    2. **Check command output (CLI-01):**
    ```rust
    #[derive(Serialize)]
    pub(crate) struct CheckOutput {

        pub(crate) info: usize,
    }
    ```
    With `print_human` method that iterates diagnostics calling `diag.print_human(w)`, then prints summary line: `"{errors} errors, {warnings} warnings, {info} info"`.

    #[derive(Serialize)]

    pub(crate) struct FindMatch {

      FM-006 (label) status: current     formal-model/v17.md
