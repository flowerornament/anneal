```rust
// NodeId: newtype index into DiGraph arena
#[derive(Clone, Copy, Debug, PartialEq, Eq, Hash, Serialize)]
pub struct NodeId(u32);

impl NodeId {
    pub fn new(index: u32) -> Self { Self(index) }
    pub fn index(self) -> usize { self.0 as usize }
}

// HandleKind: the four kinds per KB-D2
#[derive(Clone, Debug, PartialEq, Eq, Hash, Serialize)]
pub enum HandleKind {
    File(Utf8PathBuf),
    Section { parent: NodeId, heading: String },
    Label { prefix: String, number: u32 },
    Version { artifact: NodeId, version: u32 },
}

// Handle: the triple (identity, kind, state) per KB-D1
#[derive(Clone, Debug, Serialize)]
pub struct Handle {
    pub id: String,           // unique identity string
    pub kind: HandleKind,
    pub status: Option<String>,  // frontmatter status value (None = no frontmatter)
    pub file_path: Option<Utf8PathBuf>,  // source file for non-File handles
    pub metadata: HandleMetadata,
}

#[derive(Clone, Debug, Default, Serialize)]
pub struct HandleMetadata {
    pub updated: Option<chrono::NaiveDate>,
    pub superseded_by: Option<String>,
    pub depends_on: Vec<String>,
    pub discharges: Vec<String>,
    pub verifies: Vec<String>,
}
```

Use `camino::Utf8PathBuf` for all paths. Derive `Serialize` on all public types. Do NOT use `unwrap()` anywhere.

Create `src/graph.rs` with the hand-rolled DiGraph per spec section 5 (KB-D5) and section 15.3:

```rust
use crate::handle::{Handle, NodeId};

#[derive(Clone, Copy, Debug, PartialEq, Eq, Serialize)]
pub enum EdgeKind {
    Cites,
    DependsOn,
    Supersedes,
    Verifies,
    Discharges,
}

#[derive(Clone, Debug, Serialize)]
pub struct Edge {
    pub source: NodeId,
    pub target: NodeId,
    pub kind: EdgeKind,
}

pub struct DiGraph {
    nodes: Vec<Handle>,
    fwd: Vec<Vec<Edge>>,   // fwd[src] = outgoing edges from src
    rev: Vec<Vec<Edge>>,   // rev[dst] = incoming edges to dst
}
```

Implement DiGraph methods:
- `pub fn new() -> Self` -- empty graph
- `pub fn add_node(&mut self, handle: Handle) -> NodeId` -- appends to arena, returns index, grows fwd/rev
- `pub fn add_edge(&mut self, source: NodeId, target: NodeId, kind: EdgeKind)` -- appends edge to both fwd[source] and rev[target]
- `pub fn node(&self, id: NodeId) -> &Handle` -- index lookup
- `pub fn node_mut(&mut self, id: NodeId) -> &mut Handle` -- mutable index lookup
- `pub fn node_count(&self) -> usize`
- `pub fn edge_count(&self) -> usize` -- sum of all fwd edges
- `pub fn outgoing(&self, id: NodeId) -> &[Edge]` -- fwd adjacency
- `pub fn incoming(&self, id: NodeId) -> &[Edge]` -- rev adjacency
- `pub fn edges_by_kind(&self, id: NodeId, kind: EdgeKind) -> Vec<&Edge>` -- filtered outgoing edges (per spec: typed traversal as first-class API)
- `pub fn nodes(&self) -> impl Iterator<Item = (NodeId, &Handle)>` -- enumerate all nodes with their IDs

Per GRAPH-05: all 5 EdgeKind variants are defined. Per GRAPH-06: the graph is a plain struct with no persistence logic -- ephemeral, recomputed each invocation. Per D-01: all edge kinds are present for Phase 1 inference.
