# Harbor Ledger Sample Corpus

Harbor Ledger is a fictional offline coordination system for field teams that
track water stations, supply crates, and inspection notes while disconnected
from the network. The corpus is intentionally realistic enough to exercise
anneal's graph extraction, but it contains no private project content.

The current spine is FM-17, implemented by IMP-17 and reviewed by REV-17. The
open decisions OQ-17, OQ-18, and OQ-19 keep the fixture active.
