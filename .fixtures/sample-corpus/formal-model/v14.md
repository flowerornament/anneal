# FM-14 Route Notebook Baseline

FM-14 treated every station visit as a plain notebook entry. It was useful for
early pilots, but it could not distinguish a planned route from an observed
route. FM-15 supersedes this draft.

## Constraints

The model recorded crates, crews, and stations, but it had no independent
verification trail for REV-17 style audits.
