# FM-17 Field Evidence Model

FM-17 is the current route ledger model for Harbor Ledger. It keeps station
observations, crate transfers, and review notes in one evidence graph while
preserving the order in which field teams actually observed events.

## Summary

The model resolves OQ-18 by requiring either a receiver scan or two independent
witness notes. It keeps OQ-17 open because delayed station reports still need
a policy for choosing between two plausible observation times.

## Method

Every route event records a crew, a station, an observation time, and an
evidence class. Crate transfers are linked to the route event that created the
physical handoff. Reviews cite REV-17 when the conformance audit confirms the
model.

## Consequences

IMP-17 implements FM-17. The rollout plan depends on this file and on the
conformance audit.
